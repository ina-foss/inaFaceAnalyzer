## Installation

```
apt-get install cmake
```

Before installation, you should install a fork of keras-vggface
```
git clone git@github.com:DavidDoukhan/keras-vggface.git
cd keras-vggface
pip install .
```
Then return to the base dir and
```
pip install .
```
